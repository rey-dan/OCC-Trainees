HTML

  1. Doctype - Indicates the type of html being used
  2. html - Defines the beginning and ending of a html file
  3. head - Declare customization and other related elemnts for the webpage
  4. title - Indicates the title of the webpage
  5. body - Container for all the HTML elements
  6. h1 - Holds text in large font
  7. p - Holds a paragraph
  8. img - Holds an image file
  9. a - Use to add link

CSS

  1. Inline - CSS written along with the htnl elements
  2. Internal - CSS decalred within the webpage' header that is enclosed in style tag
  3. External - CSS declared on a separate file
  4. id - #
  5. class - .
  6. all - *
  7. pseudo selector - Element's state
  8. position property - location of elements in a webpage
